# BDR Test

## **Resumo**

Desenvolvimento do front conforme layout.

## **Tecnologias Utilizadas**

- Gulp
- Sass
- Javascript

## **Instalação das dependências**

Para instalar as dependências, deve se ter a versão mais atualizada do npm e digitar:

```
npm install
```

## **Execução do projeto**

Para execução do projeto

```
gulp
```
